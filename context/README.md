# CONTEXT

This folder holds manual context for ADF-V2 while the foundation is being defined.

Current rule:
- if context matters, write it here or in another explicit repo file
- do not rely on Brain or other external memory tools as the authority for v2 startup truth
- markdown files in this folder must use `UPPERCASE-WITH-HYPHENS.md`

Canonical files in this folder are:
- `context/HANDOFF.md`
- `context/OPEN-ISSUES.md`
- `context/STATUS.md`
- `context/artifacts/README.md`
- `context/decisions/README.md`
