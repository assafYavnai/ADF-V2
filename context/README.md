# CONTEXT

This folder holds manual context for ADF-V2 while the foundation is being defined.

Current rule:
- if context matters, write it here or in another explicit repo file
- do not rely on Brain or other external memory tools as the authority for v2 startup truth
- governed created entry files in this folder must use `UPPERCASE-WITH-HYPHENS.md`
- fixed canonical files in this folder may use their established repo-truth names such as `HANDOFF.md`, `OPEN-ISSUES.md`, `STATUS.md`, and `README.md`

Canonical files in this folder are:
- `context/HANDOFF.md`
- `context/OPEN-ISSUES.md`
- `context/STATUS.md`
- `context/artifacts/README.md`
- `context/decisions/README.md`

`context/HANDOFF.md` carries:
- the narrative current-state handoff
- the curated decision files the next agent must read for that handoff

The handoff decision list is curated.
It is not a flat summary of every decision in the repo.
