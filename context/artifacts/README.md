# ARTIFACTS

This folder holds pre-canonical draft artifacts for ADF-V2 while the foundation is being defined.

Rule:
- drafts may live here
- canonical files may also remain in their final location while still marked `DRAFT`
- nothing here is treated as frozen system truth until freeze approval is completed under the global freeze rule
- pre-canonical artifact filenames must use the governed entry-file naming rule: `UPPERCASE-WITH-HYPHENS.md`
- pre-canonical artifacts in this folder should carry `STATUS`, `OWNER`, `FREEZE-GATE`, and `PROMOTED-TARGET`

This folder readme does not define a separate local freeze rule.
Artifact freeze is governed by `CONTEXT.md` and therefore requires approval from both CEO and CTO.
