# DECISIONS

This folder holds frozen decisions for ADF-V2.

Rule:
- no decision is frozen unless it is explicitly approved by the CEO
- agents must not freeze system-defining decisions by inference
- decision filenames must use `UPPERCASE-WITH-HYPHENS.md`
- each decision entry must contain `CONTEXT`, `OPTIONS`, `DECISION`, and `APPROVAL`
- save a decision when a real fork is closed by the CEO or when the CEO explicitly asks to save it
- files in this folder hold frozen decisions only
- `APPROVAL` in this folder is `EXPLICIT`
- implicit agent assumptions do not belong in this folder until the CEO explicitly approves them as real decisions
